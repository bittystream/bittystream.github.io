    计网-物理层 小小结

    Bitty的博客
    2020-09-17

    Some terminology

    1. 数据(data)

    数据是运送信息的实体

    2. 信号(signal)

    数据的电气/电磁表现

    3. 模拟(analog)信号

    连续变化的信号，例如用一系列连续变化的电磁波(如无线电与电视广播中的电磁波)，或电压信号(如电话传输中的音频电压信号)

    4. 数字(digital)信号

    取值为有限的离散值的信号，例如用一系列断续变化的电压脉冲(高低电平；如我们可用恒定的正电压表示二进制数1，用恒定的负电压表示二进制数0)，或光脉冲

    5. 模拟数据

    取值为连续值的数据；采用模拟信号表示的数据。模拟数据是指在某个区间产生的连续值，例如，声音、图像、温度、压力。

    6. 数字数据

    取值为不连续数值的数据；采用数字信号表示的数据。

    7. 基带(baseband)信号

    发送端发出的没有经过调制的原始电信号

    8. 带通信号/频带(passband)信号

    基带信号经过载波调制后，频率范围搬移到较高频段的信号

    9. 码元(symbol)

    在使用时间域的波形表示数字信号时，代表不同离散数值的基本波形

    10. 单工(simplex)通信

    只有一个方向、没有反方向的交互的通信
    e.g. 广播

    11. 半双工(half-duplex)通信

    通信双方都可以发送消息，但不能双方同时发送或接收消息的通信
    e.g. 对讲机

    12. 全双工(full-duplex)通信

    通信双方可以同时发送和接收消息的通信
    e.g. 电话

    13. 串行传输

    传输中只有一个数据位，每位数据占据固定时间长度的传输

    14. 并行传输

    传输中有多个数据位同时在设备之间进行的传输
    数据在计算机多采用并行传输方式，但是在通信线路上一般都采用串行传输方式。

    15. 信道(channel)

    基于物理传输的媒介。信号传输的媒介。

    16. 传输媒介(medium)

    传输媒体并不是物理层。传输媒体在物理层的下面。物理层需要考虑的是怎样才能在连接各种计算机的传输媒介上传输数据比特流，而不是指具体的传输媒介。

    17. 多路复用(multiplexing)

    多路复用是指以同一传输媒质（线路）承载多路信号进行通信的方式。各路信号在送往传输媒质以前，需按一定的规则进行调制，以利于各路已调信号在媒质中传输，并不致混淆，从而在传到对方时使信号具有足够能量，且可用反调制的方法加以区分、恢复成原信号。

    18. 载波(carrier)

    载波或者载频（载波频率）是一个物理概念，是一个特定频率的无线电波，单位Hz，是一种在频率、幅度或相位方面被调制以传输语言、音频、图象或其它信号的电磁波。

    四个特性

    反映在 物理接口协议中 的物理接口的特性：机械特性、电气特性、功能特性与规程特性。

    通信信道的分类

    1. 按传输媒介分

    按传输媒介分类，通信信道可分为有导信道（电话线、双绞线、同轴电缆、光纤等）和无线信道（卫星、微波、无线电）

    2. 按多路复用方式分

    (1) 频分复用(FDM)
    频分复用按频谱划分信道，多路基带信号被调制在不同的频谱上。因此它们在频谱上不会重叠，即在频率上正交，但在时间上是重叠的，可以同时在一个信道内传输。

    (2) 时分复用(TDM)
    时分复用即将一条物理信道按时间分成若干个时间片轮流地分配给多个信号使用。

    (3) 波分复用和码分多址(CDMA)
    光的波分多路复用是指在一根光纤中传输多种不同波长的光信号，由于波长不同，所以各路光信号互不干扰，最后再用波长解复用器将各路波长分解出来。
    码分多址的原理：给基站分配两两正交的编码，在接收端利用码型的正交性，通过地址识别(相关检测)，从混合的信号中选出相应的信号。

    3. 按传输的信号种类分

    Just a quick reminder，信道是传输信号的媒介！
    (1) 模拟信道：传输模拟信号
    (2) 数字信道：传输数字信号

    5. 按共享类型分

    数据与信号

    数据分为模拟数据和数字数据，信号分为模拟信号和数字信号。数据是信息存储的形式，信号是信息发送的载体。在物理层，需要通过一些操作将数据转变为信号，从而在信道上传播。

    1. 模拟数据 -> 模拟信号

    方法：载波调制

    2. 模拟数据 -> 数字信号

    方法：脉冲编码调制(Pulse Code Modulation, PCM)
    PCM有三个步骤：采样、量化、编码

    3. 数字数据 -> 模拟信号

    方法：调制
    there should be a pic. showing AM FM PM

    4. 数字数据 -> 数字信号

    方法：通过数字发送器(digit emitter)进行编码变换
    编码方式（部分）：NRZ(Non-Return-to-Zero, 不归零)、NRZI(Non-Return-to-Zero_Invert-on-ones, 不归零对1反向)、双相位、曼彻斯特、差分曼彻斯特

    数据在信道上的传输

    模拟信号只能在模拟信道上传输，数字信号只能在数字信道上传输，因此对于收到的数据，需要先将其转化为对应信道的信号，然后进行传输，最后还需转化为与发送的相同的数据类型。两两组合又可分为四种：

    1. 模拟数据通过模拟信道传输

    2. 数字数据通过模拟信道传输

    3. 模拟数据通过数字信道传输

    4. 数字数据通过数字信道传输
